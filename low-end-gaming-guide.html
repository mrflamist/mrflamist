Running modern games on ancient hardware?

Written by Jackie N.S.F. - 3rd of February 2022

Running ancient games on modern hardware is very easy. Just download an emulator or set up a virtual machine and you're golden.
Running modern games on ancient hardware? Not so much. Either it won't work at all, or it'll stutter so much, you'll be happy to
even see the main menu. So what can you do? Buy a new PC! Easy enough! But is it really?

No. It isn't. There's currently a microchip shortage, scalpers are scalpier than ever and COVID is still present. If new hardware is
not an option, we'll have to make do with what we have... unless you're planning on selling your organs or family members.

THE GUIDE:

This guide is mostly intented for systems running Windows 7 SP1 or later and will not work for all games!
Whatever I tell you to do here should be safe, but you should make backups regardless.
I'm not responible for any damage you do to your installation/machine.

-Disable any unecessary startup software.

On Windows 7, press Win + R, type msconfig, press OK/Enter, switch to the startup tab, disable everything that doesn't have
to be running on startup. Be careful with what you disable, there could be drivers and such. If you're not sure, look it up.

-Keep all unnecessary software closed. Try looking for lighter alternatives to the software you need.

Third party clients can, for example, solve the electron mess the Discord is. It's against TOS though.
I can offer you some alternatives to Discord though!

I myself use Discord on phone and if I have to use it on my laptop, I use it in the browser.
I am however using AIM and MSN Messenger when not using Discord. Together they use about 44 MB of RAM.

-Lower your game's settings or use retail version of the game.

The less stuff your game has to load, the less RAM it needs. Don't be scared of editing config files. Just make a backup first.

-Disable visual effects.

Righ-click on This Computer, select Properties, then click on Advanced system settings on the left panel, select
Performance and then disable the effects you don't need. For maximum performance, select Adjust for best performance.

-Use only one Antivirus.

If you're clever enough, you don't even need one. However, Windows Defender should suffice.

-Defragment your hard drive regularly

If you're not on Windows XP, your system should be defraging automatically. It doesn't hurt to check though. I recommend UltraDefrag as it takes care of what Windows doesn't.
